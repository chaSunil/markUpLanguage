# markUpLanguage
마크업 언어 사용법 정리

Readme 파일을 작성하기 위해서 정리해놓는 마크업 언어 사용법 정리입니다.

### 제목
.#의 갯수에 따라서 html의 h1 태그와 동일하게 사용이 가능하다.
ex) h1 제일 큼 h2 그 다음으로 큰..
